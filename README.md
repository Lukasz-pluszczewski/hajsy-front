# Simple cash app

NOTE: This repo will most likely not bring you any value.

#### Clone the project
```bash
git clone git@github.com:Lukasz-pluszczewski/hajsy-front.git
cd hajsy-front
```

#### Install dependencies
```bash
npm i
```

#### Run development server
```bash
npm run dev
```

or

#### Build the project
```bash
npm run build
```

#### Run the app
```bash
npm start
```
